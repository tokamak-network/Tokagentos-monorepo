Deploy AI agents as serverless AWS Lambda functions.

## Quick Start

```bash
cd examples/aws
export OPENAI_API_KEY="your-key"

# TypeScript
cd typescript && bun run build
sam deploy --guided --parameter-overrides OpenAIApiKey=$OPENAI_API_KEY
```

## Architecture

```
┌──────────────┐     ┌─────────────────┐     ┌────────────────┐
│  Client      │────▶│  API Gateway    │────▶│  Lambda        │
│              │◀────│  (HTTP API)     │◀────│  (tokagentOS)     │
└──────────────┘     └─────────────────┘     └────────────────┘
```

## TypeScript Handler

```typescript
import { APIGatewayProxyHandler } from "aws-lambda";
import { AgentRuntime } from "@tokagentos/core";
import { openaiPlugin } from "@tokagentos/plugin-openai";

let runtime: AgentRuntime | null = null;

export const handler: APIGatewayProxyHandler = async (event) => {
  if (!runtime) {
    runtime = new AgentRuntime({
      character: { name: "Tokagent", bio: "A helpful AI." },
      plugins: [openaiPlugin],
    });
    await runtime.initialize();
  }

  const { message } = JSON.parse(event.body || "{}");
  const response = await runtime.useModel("TEXT_LARGE", { prompt: message });

  return {
    statusCode: 200,
    body: JSON.stringify({ response: String(response) }),
  };
};
```

## Configuration

| Variable         | Default       | Description    |
| ---------------- | ------------- | -------------- |
| `OPENAI_API_KEY` | Required      | OpenAI API key |
| `CHARACTER_NAME` | Tokagent         | Agent name     |
| `CHARACTER_BIO`  | A helpful AI. | Agent bio      |
| `LOG_LEVEL`      | INFO          | Logging level  |
